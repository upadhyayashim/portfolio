## ⚙️ Local Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/upadhyayashim/portfolio.git
cd portfolio
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## 🔐 Environment Variables

### Backend (.env)

```
PORT=5000
MONGO_URI=your_mongodb_atlas_uri
CLOUDINARY_CLOUD_NAME=xxxx
CLOUDINARY_API_KEY=xxxx
CLOUDINARY_API_SECRET=xxxx
JWT_SECRET=xxxx
```

### Frontend (.env)

```
VITE_API_URL=http://localhost:5000/api
```
